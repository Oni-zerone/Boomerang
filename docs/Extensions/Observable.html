class Observable<Element> : ObservableType
  • structured()

    Creates an observable that emits a ModelStructure containg the collections’ elements if the collection is an array of ModelType objects. Basically, it’s the same of

    let observable:Observable<[ModelType]> = ...
let structured = observable.map { ModelStructure($0) }
